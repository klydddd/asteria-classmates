Mermaid flowchart

Vertical pipeline

The review gate protects every downstream artifact: unapproved clips are excluded before split generation, benchmark evaluation, and model training.

The canonical order is Import, Normalize, Review, Build, Transcribe, Evaluate, and optional Fine-tune. Build never includes unapproved clips.

Stage details

What each stage produces

These are the concrete pipeline responsibilities behind the diagram.

Import and validation

Parses PLD sessions, matches WAV files to transcript rows, checks audio quality, and writes deterministic IDs such as pam_000001.wav.

Normalize and review

Normalizes transcript text and lets reviewers approve, reject, skip, or flag clips from a resumable terminal workflow.

Build dataset

Filters to approved clips and writes speaker-aware train, validation, and test splits with reproducible seeds.

Transcribe and evaluate

Runs ASR over a single file or split, then computes WER and CER with normalized scoring text.

Fine-tune and compare

Optionally fine-tunes Whisper with LoRA, evaluates the model, and compares baseline vs fine-tuned results.

Package documentation

Generates dataset cards, model cards, JSON reports, Markdown reports, and comparison summaries.

Command path

End-to-end commands

This is the shortest local path that corresponds to the diagram.

Pipeline command sequence

copy-ready 
bosesph import-pld PLD/PAM/0400 --output outputs/dataset
bosesph validate-metadata outputs/dataset/metadata.csv
bosesph normalize-transcripts outputs/dataset
bosesph review outputs/dataset
bosesph build-dataset outputs/dataset --output outputs/dataset_15spk
bosesph transcribe outputs/dataset_15spk \
  --model openai/whisper-small --language tl \
  --split test --output outputs/benchmark/baseline_predictions.csv
bosesph evaluate \
  --predictions outputs/benchmark/baseline_predictions.csv \
  --output outputs/benchmark/baseline \
  --model-name "Baseline Whisper Small" \
  --language kapampangan
bosesph finetune outputs/dataset_15spk \
  --output outputs/model/bosesph-kapampangan-v1 \
  --base-model openai/whisper-small \
  --language tl
bosesph compare \
  --baseline outputs/benchmark/baseline/results.json \
  --finetuned outputs/benchmark/finetuned/results.json \
  --output outputs/benchmark/comparison.md

Artifact map

Final outputs

The diagram terminates in reusable dataset, benchmark, model, and documentation artifacts.

Dataset

  • audio/ standardized WAV files
  • metadata.csv
  • train.csv, validation.csv, test.csv
  • dataset_stats.json and dataset_card.md

Benchmark

  • Baseline and fine-tuned predictions CSVs
  • results.json files with WER and CER
  • report.md summaries
  • comparison.md

Model

  • Fine-tuned model and processor files
  • training_config.json
  • model_card.md
  • Optional Colab training and evaluation notebooks
